🐛 TROUBLESHOOTING

  Port in use?
    lsof -i :3000  # or :5000
    kill -9 <PID>

  npm install fails?
    rm -rf node_modules package-lock.json
    npm install

  Database errors?
    rm backend/alerts.db
    cd backend && npm run seed

  CORS issues?
    Check FRONTEND_URL in backend/.env
